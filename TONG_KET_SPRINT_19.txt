Yêu cầu: Cần giọng nữ chuyên nghiệp hơn giọng Google Translate hiện tại.

Hiện tại: Google Translate TTS (giọng robot) + Web Speech API (fallback)

5 phương án đề xuất:

┌──────────────────────────────────────────────────────────────────────────────┐
│ #  │ Phương án              │ Giọng nữ Việt        │ Miễn phí     │ Chất lượng │
├──────────────────────────────────────────────────────────────────────────────┤
│ 1  │ Google Cloud TTS       │ vi-VN-Neural2-A      │ 1M chars/mo  │ ⭐⭐⭐⭐    │
│ 2  │ Azure Speech           │ vi-VN-HoaiMyNeural   │ 500K chars/mo│ ⭐⭐⭐⭐⭐  │

│ 4  │ Zalo AI                │ Nhiều giọng nữ       │ Free tier    │ ⭐⭐⭐      │
│ 5  │ Edge TTS (miễn phí)    │ vi-VN-HoaiMyNeural   │ Unlimited    │ ⭐⭐⭐⭐⭐  │
└──────────────────────────────────────────────────────────────────────────────┘

Khuyến nghị:
  → Edge TTS: Miễn phí 100%, giọng HoaiMyNeural cực hay, tích hợp nhanh
  → Azure Speech: Ổn định nhất, giọng Việt tốt nhất, free 500K chars/tháng
  → Google Cloud TTS: Free 1M chars/tháng, ổn định
