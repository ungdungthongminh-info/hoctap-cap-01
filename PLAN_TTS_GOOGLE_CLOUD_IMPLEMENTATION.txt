PHASE 1 - BACKEND TTS INFRASTRUCTURE

1.1. Cap nhat dependency backend
- File: backend/package.json
- Them package: @google-cloud/text-to-speech

1.2. Them env va bao mat
- File: backend/.env.example hoac file env mau dang dung
- Them cac bien:
  - TTS_PROVIDER=google-cloud
  - APP_CONTENT_VERSION=2026-04-27-v1
  - GOOGLE_APPLICATION_CREDENTIALS=./secrets/google-tts-service-account.json
  - GOOGLE_TTS_DEFAULT_LANGUAGE=vi-VN
  - GOOGLE_TTS_DEFAULT_VOICE=vi-VN-Wavenet-A
  - GOOGLE_TTS_AUDIO_ENCODING=MP3
  - TTS_MONTHLY_CHAR_LIMIT=1000000
  - TTS_MAX_TEXT_LENGTH=1800
- File: .gitignore va backend/.gitignore
  - Ignore backend/secrets/
  - Ignore file credential json

1.3. Them bang cache TTS vao backend DB
- File: backend/src/db/init.ts
- Them CREATE TABLE IF NOT EXISTS tts_cache (...)
- Cot toi thieu:
  - id
  - provider
  - voice
  - speed
  - lang
  - usage
  - content_version
  - text_hash
  - audio_path
  - char_count
  - hit_count
  - created_at
  - updated_at
- Tao unique index theo cache key:
  provider + voice + speed + lang + usage + content_version + text_hash

1.4. Tao adapter Google Cloud
- File moi: backend/src/services/tts/GoogleTtsAdapter.ts
- Nhiem vu:
  - Khoi tao client @google-cloud/text-to-speech
  - Nhap text, lang, voice, speed
  - Tra MP3 buffer
  - Validate TTS_MAX_TEXT_LENGTH

1.5. Tao cache repository
- File moi: backend/src/services/tts/cacheRepo.ts
- Nhiem vu:
  - Tao hash text
  - Tim cache theo full cache key
  - Save metadata vao sqlite
  - Tang hit_count khi cache hit
  - Tao duong dan luu file MP3 vao backend/data/tts-cache/google-cloud/<APP_CONTENT_VERSION>/
  - Co helper xoa cache neu can

1.6. Tao controller
- File moi: backend/src/controllers/ttsController.ts
- API chinh:
  - POST /tts/synthesize
    Input: text, lang, voiceId, speed, usage, contentVersion
    Output: audioUrl, cacheStatus, provider, voiceId
  - GET /tts/cache-stats
    Output: tong so file cache, usage theo ngay/thang
  - DELETE /tts/cache
    Output: xoa cache theo filter hoac toan bo cache
- Validation:
  - Text length
  - VoiceId hop le
  - Speed trong range cho phep
  - usage nam trong danh sach cho phep

1.7. Tao route
- File moi: backend/src/routes/tts.ts
- Mount cac endpoint tu controller

1.8. Noi route vao backend
- File: backend/src/index.ts
- Them app.use(API_PREFIX, ttsRoutes)
- Them static serving cho audio cache neu can:
  - /tts-audio -> backend/data/tts-cache

PHASE 2 - SAFETY, RATE LIMIT, BUDGET

2.1. Rate limit
- Gioi han request theo IP cho /api/v1/tts/synthesize
- Chan spam request lap lai qua nhanh

2.2. Text limit
- Khong cho gui text tu do qua dai
- Limit theo TTS_MAX_TEXT_LENGTH
- Khuyen nghi phia frontend chi gui text da duoc app tao ra

2.3. Log usage
- Log so ky tu da dung theo ngay va theo thang
- Co check TTS_MONTHLY_CHAR_LIMIT truoc khi goi Google Cloud
- Neu vuot limit:
  - Khong goi Google Cloud nua
  - Fallback native
  - Tra canh bao cho frontend
