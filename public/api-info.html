🤖 Dịch Vụ AI Tích Hợp

Ứng dụng Học Hùng Khởi hỗ trợ 3 nhà cung cấp AI hàng đầu.

✅ Người dùng tự quản lý API key — Hoàn toàn riêng tư
🔵

Google Gemini

Tốc độ nhanh, miễn phí ở quota cơ bản. Phù hợp nhất cho học sinh tiểu học.

Model mặc định gemini-2.0-flash
Endpoint generativelanguage.googleapis.com
Xác thực API Key trong query string
gemini-2.0-flash ⭐ gemini-1.5-flash gemini-1.5-pro
🔑 Lấy API key miễn phí
🟢

GPT – OpenAI

Mô hình nổi tiếng nhất thế giới. Tương thích với mọi API proxy trung gian.

Model mặc định gpt-4o-mini
Endpoint api.openai.com/v1
Xác thực Bearer Token (Authorization header)
gpt-4o-mini ⭐ gpt-4o gpt-3.5-turbo
🔑 Lấy API key OpenAI
🟣

Grok – xAI

AI của xAI (Elon Musk). Giao thức tương thích OpenAI, dễ tích hợp proxy.

Model mặc định grok-3-mini
Endpoint api.x.ai/v1
Xác thực Bearer Token (Authorization header)
grok-3-mini ⭐ grok-3
🔑 Lấy API key xAI

⚙️ Cách Hoạt Động

  • Người dùng mở Cài Đặt AI trong ứng dụng, nhập API key của nhà cung cấp họ chọn.
  • Mỗi tin nhắn được gửi thẳng từ trình duyệt tới server của nhà cung cấp — không qua server trung gian.
  • API key lưu ở localStorage trên thiết bị, không gửi về bất kỳ server nào của ứng dụng.
  • Người dùng có thể tùy chỉnh endpoint để dùng API proxy riêng hoặc endpoint bán lại.
  • Lịch sử hội thoại tối đa 20 lượt, tự xóa khi đóng ứng dụng.
💡 Dùng API Proxy / Bán API?
Ứng dụng hỗ trợ thay endpoint tùy ý. Nếu bạn cung cấp dịch vụ API trung gian (GPT/Gemini giá rẻ hơn), người dùng chỉ cần nhập endpoint của bạn vào ô API Endpoint trong Cài Đặt AI — không cần thay đổi gì khác.

📦 Dữ Liệu JSON

Manifest JSON đầy đủ có thể fetch bởi web tổng để đồng bộ thông tin dịch vụ tự động.

📄 /ai-services.json
fetch('/ai-services.json').then(r => r.json()).then(data => console.log(data.providers))
